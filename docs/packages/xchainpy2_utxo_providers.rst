XChainPy UTXO Data providers package
====================================

.. warning::
    This package is currently not used in the XChainPy project.
    bitcoinlib is used instead as the UTXO data provider for the XChainPy Bitcoin, Litecoin and Dogecoin clients.

UTXO Data providers enable the XChainPy clients to do the following tasks:

* Get UTXOs for an address.
* Transaction list for an address.
* Transaction details for a transaction hash.
* Broadcast a transaction.
